# 🧮 BRUTE — Neo-Brutalist Web Calculator

An advanced calculator built as part of the **InternGrow Frontend Development Track**, featuring a bold neo-brutalist design with Standard, Scientific, and Programmer modes.

## Features

- ➕ **Standard Mode** — full arithmetic: add, subtract, multiply, divide, percentage, sign toggle
- 🔬 **Scientific Mode** — sin, cos, tan, √, x², xʸ, log, ln, 1/x, x! (factorial), π, e
- 💻 **Programmer Mode** — switch between Decimal, Hexadecimal, and Binary, with AND / OR / XOR bitwise operations
- 🌗 **Dark Mode** — full light/dark theme toggle
- 🎨 **Neo-Brutalist UI** — bold borders, hard drop shadows, high-contrast colors
- 📱 Fully responsive across mobile and desktop

## Tech Stack

- HTML5
- CSS3 (custom properties, grid layout, transitions)
- Vanilla JavaScript (no frameworks or libraries)

## How to Use

1. Open `index.html` in any modern browser
2. Switch between **Standard / Scientific / Programmer** tabs at the top of the panel
3. In Programmer mode, select **DEC / HEX / BIN** to change number base
4. Toggle **DARK** in the top-right corner for dark mode
